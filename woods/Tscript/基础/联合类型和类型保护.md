# 联合类型和类型保护

## 类型保护的方式

### 类型断言 as

```
interface ming {
  name: string;
  say: () => {};
}
interface zhang {
  name: string;
  run: () => {};
}
function getPerson(person: zhang | ming) {
  if (person.name === 'ming') {
    (person as ming).say();
  }
  (person as zhang).run();
}
```

### in 语法

```
function getPerson1(person: zhang | ming) {
  if ('say' in person) {
    person.say();
  } else {
    person.run();
  }
}
```
