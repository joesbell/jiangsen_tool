const str:string='hello'

# 对象类型
 ## 数组类型
```
    const arr:number[]=[1,2,4]
```
---
 ## 对象类型
```
    const obj:{
        name:string,
        age:number
    }={
        name:'hellowrold',
        age:134
    }
```

## 函数类型
```
    const baseFun:(age:number)=>number=(age)=>{
        return age
    }
```
---
```
    const fun1=(age:number)=>{
        return age
    }
```
*正常情况下，用这种形式声明函数，只需要写入参的类型注解，返回参数没必要写，因为可以自动进行类型推断*
